Output File Name Format:

CategoryName-QueryNumberReplacementText-PageNumber

	CategoryName: Name of the category to which that page belongs (CategoryName for Query11 - Query20: ALL)
	QueryNumberReplacementText: Please check Below.
	PageNumber: Print only page number like -page1, -page2 (Not like page1 of 3, page2 of 3.....). Just page number sufficient.


QueryNumberReplacementText:

Query1 replace with: loss-makers-today-on-bse
Query2 replace with: loss-makers-last-20-days-on-bse
Query3 replace with: popular-funds-on-bse
Query4 replace with: trending-funds-on-bse
Query5 replace with: highest-rated-funds-on-bse
Query6 replace with: newly-launched-funds-on-bse
Query7 replace with: overall-best-funds-on-bse
Query8 replace with: most-expensive-mutual-funds-on-bse
Query9 replace with: cheapest-mutual-funds-on-bse
Query10 replace with: trending-new-funds-on-bse
Query11 - Query20: Same as above respectively.


Examples:

debt-fund-loss-makers-today-on-bse-page1
debt-fund-popular-funds-on-bse-page3
all-loss-makers-today-on-bse-page1
